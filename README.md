# Space Invaders 👾

## Descrição

Este projeto é uma implementação do clássico jogo **Space Invaders** desenvolvida em Python utilizando a biblioteca PPlay para a discplina de Lab de Jogos da UFF.

## Funcionalidades

- Movimentação da nave
- Disparo de projéteis
- Sistema de colisão
- Sistema de pontuação
- Ranking local
- Efeitos sonoros

## Tecnologias Utilizadas

- Python 3
- PPlay

## Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/MiguelMFR/space-invaders.git
```

2. Entre na pasta do projeto:

```bash
cd space-invaders
```

3. Execute o jogo:

```bash
python main.py
```

## Controles

| Tecla | Ação |
|--------|--------|
| A / ← | Mover para a esquerda |
| D / → | Mover para a direita |
| Espaço | Atirar |
| ESC | Voltar ao menu |
